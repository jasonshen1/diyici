## 一、每日运营流程（Daily Routine）

### ⏰ 上午 9:00-11:00

**9:00-9:30 数据复盘**
```
□ 查看昨日笔记数据（小眼睛、点赞、收藏、评论）
□ 记录爆款笔记，分析原因
□ 查看粉丝增长情况
□ 记录竞品动态（刷10分钟同行笔记）
```

**9:30-10:30 内容创作**
```
□ 拍摄/修图（封面图+内页图）
□ 撰写文案（按模板填充）
□ 添加话题标签（参考热门标签库）
```

**10:30-11:00 发布&互动**
```
□ 发布笔记（选择流量高峰期：11:00-12:00 或 20:00-21:00）
□ 评论区互动（前30分钟密集回复）
□ 私信处理（商务合作筛选）
```

### ⏰ 下午 14:00-17:00

**14:00-15:00 内容储备**
```
□ 拍摄下周内容素材
□ 整理绘本实物（按主题分类）
□ 撰写3-5篇文案草稿
```

**15:00-16:00 社群运营**
```
□ 粉丝群互动（回答问题）
□ 收集用户痛点（下次内容选题）
□ 预告明日内容
```

**16:00-17:00 商务对接**
```
□ 品牌方沟通
□ 选品测评
□ 合作方案撰写
```

### ⏰ 晚上 20:00-22:00

**20:00-21:00 直播/视频**
```
□ 直播讲绘本（每周2-3场）
□ 或拍摄视频内容
□ 直播后数据复盘
```

**21:00-22:00 学习&优化**
```
□ 刷小红书找灵感（30分钟）
□ 学习新功能（小红书更新）
□ 规划明日内容
```

## 二、内容生产SOP

### 2.1 选题流程

**周一：选题会**
```
Step 1: 数据分析（30分钟）
  - 查看上周哪些笔记表现好
  - 分析评论区用户最关心的问题
  - 记录竞品爆款选题

Step 2: 确定本周主题（30分钟）
  示例：
  周一：绝版绘本推荐
  周三：3岁必读清单
  周五：审美培养绘本
  周日：情感类绘本

Step 3: 具体选题（30分钟）
  按模板填写：
  - 主题：______
  - 痛点：______
  - 情绪词：______
  - 预期互动：______
```

### 2.2 文案撰写模板

**模板A：痛点+解决方案型**
```
【标题公式】
痛点词 + 数字 + 承诺
例：太费妈⚠️但这5本绘本一生推，3岁娃看了300遍

【正文结构】
1. 痛点共鸣（2-3句）
   "姐妹们，是不是有这样的烦恼..."

2. 解决方案（核心内容）
   "今天哈爸推荐的这5本..."

3. 具体介绍（每本1-2句）
   "第1本《XXX》，特点是..."

4. 互动引导（1句）
   "你家娃最爱哪本？评论区告诉我"

5. 标签（5-8个）
   #绘本推荐 #3岁绘本 #费妈绘本
```

**模板B：震惊+权威型**
```
【标题公式】
数字/权威 + 惊叹词 + 悬念
例：豆瓣9.5❗️让我震撼到失语❗️确定是给3岁孩子看的❓

【正文结构】
1. 震惊开场（1句）
   "看完这本绘本，我沉默了..."

2. 权威背书（1句）
   "豆瓣9.5分，绝版40年终于再版"

3. 内容介绍（3-5句）
   "这本书讲的是..."

4. 情感升华（2-3句）
   "不只是给孩子看，大人也会被治愈"

5. 购买引导（1句）
   "链接在左下角，限量发售"
```

**模板C：情感共鸣型**
```
【标题公式】
身份 + 情感词 + 稀缺性
例：终于找到了！给姥姥的专属绘本❤️读完泪崩了

【正文结构】
1. 情感钩子（2句）
   "小时候是姥姥带我长大..."

2. 产品介绍（2-3句）
   "这本《XXX》讲的就是祖孙情..."

3. 个人故事（3-5句）
   "我给儿子读的时候，想起了..."

4. 情感升华（2句）
   "有些情感，需要代代相传"

5. 行动号召（1句）
   "给家里的老人也买一本吧"
```

### 2.3 拍摄SOP

**封面图要求：**
```
□ 尺寸：3:4 竖图（小红书最佳）
□ 主体：绘本封面清晰展示
□ 文字：大字号标题，对比色
□ 风格：简洁、高级、有质感
□ 工具：美图秀秀/醒图

封面文字公式：
[数字] + [痛点/利益点] + [情绪符号]
例：5本｜费妈但一生推｜豆瓣9.5
```

**内页图要求：**
```
□ 3-9张（根据内容）
□ 实拍图+文字标注
□ 光线充足，无反光
□ 背景简洁（白桌布/木质桌面）
□ 可加入孩子阅读场景（如有）
```

### 2.4 发布 checklist

**发布前检查：**
```
□ 封面图是否清晰？
□ 标题是否有吸引力？
□ 正文是否有错别字？
□ 标签是否添加（5-8个）？
□ 定位是否添加（北京）？
□ 发布时间是否合适？
```

**最佳发布时间：**
```
工作日：11:30-12:30、20:00-22:00
周末：10:00-11:00、21:00-23:00
```

## 三、爆款打造流程

### 3.1 爆款复制SOP

**Step 1: 发现爆款（每天）**
```
□ 查看自己的爆款笔记（赞>1000）
□ 记录：标题、封面、发布时间
□ 分析：为什么爆？痛点？情绪？
```

**Step 2: 拆解爆款（每周）**
```
□ 拆解5个自己的爆款
□ 拆解5个竞品爆款
□ 制作《爆款元素清单》
```

**Step 3: 复制爆款（持续）**
```
□ 换汤不换药：同样的结构，换绘本
□ 系列化：爆款A→爆款A-2→爆款A-3
□ 升级迭代：爆款B比爆款A更好
```

## 四、变现执行SOP

### 4.1 带货流程

**选品标准：**
```
□ 豆瓣评分 8.5+ 或无评分但口碑好
□ 适合3-6岁（核心受众）
□ 有审美价值（符合账号调性）
□ 佣金比例 20%+ 或有品牌合作费
□ 出版社/品牌可靠
```

**带货笔记流程：**
```
Day 1: 选品+阅读
  □ 收到样品或购买绘本
  □ 深度阅读，记录亮点
  □ 思考：适合什么场景？解决什么痛点？

Day 2: 内容创作
  □ 撰写文案（按模板）
  □ 拍摄图片（封面+内页）
  □ 制作封面图

Day 3: 发布+跟进
  □ 上午11点发布
  □ 评论区引导购买
  □ 私信回答购买问题
  □ 记录转化数据
```

### 4.2 商务合作流程

**合作筛选标准：**
```
□ 品牌调性匹配（教育、绘本、母婴）
□ 预算符合预期（报价：粉丝数×0.5-1元）
□ 产品真实可用（亲自试用）
□ 合作条款合理（不接硬广）
```

**报价参考：**
```
16万粉丝，垂类教育博主：
- 单篇笔记：8000-15000元
- 视频笔记：15000-30000元
- 直播专场：50000-100000元
- 年度合作：面议（通常50万+）
```

**合作执行流程：**
```
Step 1: 商务洽谈（1-3天）
  □ 了解品牌需求
  □ 确认合作形式（图文/视频/直播）
  □ 报价并谈判
  □ 签署合同（预付50%）

Step 2: 内容创作（3-5天）
  □ 收到产品并体验
  □ 撰写文案（给品牌方审核）
  □ 修改并确认
  □ 拍摄制作

Step 3: 发布执行（1天）
  □ 按约定时间发布
  □ 品牌方确认
  □ 数据跟踪（7天内）

Step 4: 结算（7-30天）
  □ 提供数据报告
  □ 结算尾款
  □ 维护长期关系
```

## 六、危机处理SOP

### 6.1 负面评论处理

**Level 1: 一般质疑**
```
回复模板：
"感谢您的反馈～每个人感受不同，
这本绘本确实不是适合所有孩子，
建议可以先看看内页再决定❤️"
```

**Level 2: 恶意攻击**
```
处理方式：
1. 不回复，不纠缠
2. 截图留证
3. 举报该评论
4. 如持续攻击，拉黑处理
```

**Level 3: 品牌方投诉**
```
处理方式：
1. 立即下架相关笔记
2. 与品牌方沟通
3. 如有违约，按合同赔偿
4. 公开道歉（如涉及虚假宣传）
```

### 6.2 账号异常处理

**情况1: 限流**
```
症状：笔记曝光突然下降（<1000）
处理：
□ 检查是否违规（刷量/敏感词）
□ 停止发布，观察3天
□ 联系小红书客服
□ 后续降低发布频率，提高质量
```

**情况2: 被举报**
```
症状：收到官方违规通知
处理：
□ 查看具体违规原因
□ 申诉（如误判）
□ 删除违规内容
□ 整改后重新发布
```
