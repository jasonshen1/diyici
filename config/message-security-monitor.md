# 消息异常监控配置
# 用于检测和防止异常自动消息发送

## 监控规则

### 禁止自动发送的消息类型
- 商品推广（京东、淘宝等）
- 未经用户明确请求的推荐
- 重复性营销内容

### 允许的消息类型
- 用户明确请求的信息查询
- 系统维护通知（仅限已配置的cron任务）
- 用户设置的定时提醒

## 安全检查点

### 发送前检查
1. 确认消息是由用户触发的
2. 确认消息内容与当前session上下文相关
3. 确认不是重复发送相同内容

### 异常检测
- 如果1小时内发送超过3条未经请求的消息 → 暂停所有主动发送
- 如果检测到商品推广内容 → 立即拦截并记录
