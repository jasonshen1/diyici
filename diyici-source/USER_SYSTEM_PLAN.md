# 用户系统开发计划

## 功能需求
1. 用户注册/登录
2. JWT会话管理
3. 用户中心（查看自己的方案）
4. 权限控制（只能删除自己的内容）
5. 微信扫码登录（可选）

## 开发步骤
1. 数据库设计 - 用户表
2. 后端API - 注册/登录/JWT
3. 前端页面 - 登录/注册/用户中心
4. 权限中间件
5. 集成测试

## 预计时间
- 后端API：2-3小时
- 前端页面：2-3小时
- 调试优化：1-2小时
- 总计：半天到一天

## 技术方案
- 数据库：SQLite（扩展用户表）
- 认证：JWT + bcrypt
- 前端：原生JS + localStorage
- 登录方式：账号密码 + 微信扫码（二期）
